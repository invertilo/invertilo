## 🧑‍💻 Presentación

Tengo 19 años y soy un apasionado de la tecnología, el desarrollo de software y la ciberseguridad. Me considero una persona curiosa, autodidacta y orientada a resolver problemas complejos mediante el desarrollo de código eficiente e inteligente.

Mi viaje en la programación comenzó por la curiosidad de entender el funcionamiento interno de los sistemas. Hoy en día, dedico mi tiempo a diseñar automatizaciones optimizadas, integrar APIs para simplificar procesos y analizar la seguridad de aplicaciones web para comprender sus puntos débiles y cómo fortalecer sus defensas de manera efectiva.

## ⚡ Áreas de Especialización y Enfoque

### 🖥️ Desarrollo Backend y Datos
* **Lógica Robusta:** Creación de arquitecturas del lado del servidor escalables utilizando **JavaScript**, **TypeScript** y **Python**.
* **Gestión de Datos:** Diseño y optimización de consultas en bases de datos estructuradas con **SQL**.

### 🤖 Automatización y Flujos de Trabajo
* **Scripts a Medida:** Desarrollo de automatizaciones personalizadas para optimizar tareas repetitivas.
* **Integración Avanzada:** Consumo masivo e integración de APIs de servicios externos de forma eficiente.

### 🛡️ Seguridad Web y Análisis de Vulnerabilidades (Enfoque Ético)
* **Auditoría de APIs:** Análisis de fallos de seguridad comunes en interfaces de programación guiados por el estándar **OWASP Top 10 API Security**.
* **Protección Perimetral:** Investigación en entornos controlados/laboratorios sobre el comportamiento de sistemas WAF, mitigación de bots y mecanismos de validación (Captcha/Cloudflare) para entender cómo mejorar su configuración y robustez.
