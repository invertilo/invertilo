Tengo 19 años y soy un apasionado de la tecnología, el desarrollo de software y la ciberseguridad. Me considero una persona curiosa, autodidacta y orientada a resolver problemas complejos mediante código eficiente, estructurado y seguro.

Hoy en día, dedico mi tiempo a diseñar automatizaciones optimizadas, desarrollar y optimizar scripts avanzados en **Lua** para servidores de **FiveM** bajo un enfoque ofensivo y defensivo (creación de sistemas **Anticheat** a medida, mitigación de inyecciones de `TriggerServerEvent` y detección de conductas maliciosas/cheats), integrar bases de datos relacionales robustas como **PostgreSQL** y analizar la seguridad de aplicaciones web para comprender sus puntos débiles y cómo fortalecer sus defensas de manera efectiva bajo entornos contenedorizados con **Docker**.

## ⚡ Áreas de Especialización y Enfoque

### 🖥️ Desarrollo Backend y Datos
* **Lógica Robusta:** Creación de arquitecturas del lado del servidor escalables utilizando **JavaScript**, **TypeScript** y **Python**.
* **Gestión de Datos:** Diseño, optimización y modelado de bases de datos relacionales complejas utilizando **PostgreSQL** y **SQL**.
* **Infraestructura y Contenedores:** Modularización y despliegue de entornos de desarrollo y producción utilizando **Docker** para garantizar la portabilidad y consistencia del software.

### 🎮 Desarrollo FiveM y Scripting en Lua
* **Optimización de Recursos (Resmon):** Desarrollo de scripts optimizados en **Lua** enfocados en bajo consumo de CPU/Memoria (`resmon < 0.01ms` en idle) para mantener la estabilidad del servidor.
* **Arquitectura Cliente-Servidor:** Implementación avanzada de sincronización de estados, comunicación mediante eventos optimizados y uso eficiente de State Bags.

### 🛡️ Desarrollo de Anticheats (FiveM Security)
* **Protección contra Inyección de Eventos:** Aseguramiento y sanitización de llamadas en el backend frente a inyecciones no autorizadas de `TriggerServerEvent` mediante validaciones estrictas de origen, tokens dinámicos temporales y límites de tasa (*rate limiting*).
* **Detección de Conductas Maliciosas (Cheats):** Creación de algoritmos heurísticos cliente/servidor para identificar anomalías de movimiento (noclip, speedhack), spawns no autorizados de entidades (vehículos, objetos, armas) y manipulación de memoria/variables nativas.
* **Detección de Mod Menus y Executors:** Implementación de contramedidas activas para detectar la inyección y ejecución de scripts externos (executors de cheats) mediante el escaneo/monitoreo de tablas globales, recursos ocultos y funciones nativas alteradas.

### 🤖 Automatización y Flujos de Trabajo
* **Scripts a Medida:** Creación de herramientas personalizadas para optimizar procesos repetitivos y procesar flujos de información complejos.
* **Integración Avanzada:** Consumo eficiente y seguro de APIs de servicios externos de alto volumen.

### 🛡️ Seguridad Web y Pentesting Ético
* **Auditoría de APIs:** Identificación de vulnerabilidades guiado por el estándar **OWASP Top 10 API Security** para evitar fugas de información.
* **Protección Perimetral:** Investigación en entornos de laboratorio sobre el comportamiento de sistemas WAF, mitigación de bots y mecanismos de validación (Captcha/Cloudflare) para entender cómo mejorar su configuración y robustez.
